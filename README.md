# 🍽️ Restaurant QR Generator

Быстрый и удобный инструмент для ресторанов и кафе, позволяющий мгновенно превратить ссылку на ваше онлайн-меню в функциональный QR-код.

### 🚀 О проекте
В современном мире гостям удобнее сканировать QR-код на столе, чем листать бумажные меню. Этот микро-сервис создан для того, чтобы владелец заведения мог сгенерировать готовый код за пару секунд без регистрации, СМС и сложных настроек.

### ✨ Основные возможности
*   **Мгновенная генерация:** Введите URL вашего меню — получите QR-код в реальном времени.
*   **Легкость:** Никакого бэкенда, всё работает прямо в браузере пользователя.
*   **Адаптивность:** Идеально работает как на десктопах, так и на смартфонах персонала.
*   **Скачивание:** Возможность сохранить QR-код как изображение (PNG) для последующей печати.

### 🛠 Стек технологий
*   **HTML5 / CSS3** — адаптивная верстка с использованием современных подходов к стилизации.
*   **Vanilla JavaScript (ES6+)** — логика генерации без использования тяжелых фреймворков.
*   **QRCode.js** — проверенная библиотека для клиентской генерации QR-кодов.
