# 🍽️ Restaurant Menu Website

Адаптивный сайт-меню для ресторанов и кафе с интегрированной возможностью быстрого шеринга через QR-код.

### 🚀 О проекте
Этот проект — готовое решение для цифровизации ресторанного бизнеса. Вместо использования устаревших бумажных меню, заведение может разместить один QR-код на столах. Гости сканируют код, переходят на этот сайт и получают удобный доступ ко всем позициям: напиткам, десертам и основным блюдам.

### ✨ Основные возможности
* **Удобная навигация:** Быстрое переключение между категориями меню (напитки, десерты, основное) без перезагрузки страницы.
* **Встроенный QR-код:** На странице размещен QR-код, который ведет на этот самый сайт. Пользователи или персонал могут скачать его в формате PNG прямо с экрана, чтобы распечатать и наклеить на столики.
* **Адаптивный дизайн:** Меню одинаково красиво и четко отображается как на экранах компьютеров, так и на смартфонах гостей.
* **Максимальная скорость:** Сайт работает полностью на стороне клиента (Vanilla JS), поэтому вкладки переключаются мгновенно.

### 🛠 Стек технологий
* **HTML5 / CSS3** — чистая верстка и интуитивно понятный интерфейс.
* **Vanilla JavaScript (ES6+)** — логика переключения вкладок меню и управление интерфейсом.
* **QRCode.js** — библиотека для вывода QR-кода ссылки сайта на стороне клиента.

# 🍽️ Restaurant Menu Website

A responsive menu website for restaurants and cafes with integrated QR code sharing.

### 🚀 About the Project
This project is a ready-made solution for the digitalization of the restaurant business. Instead of using outdated paper menus, establishments can place a single QR code on tables. Guests scan the code, are redirected to this website, and gain convenient access to all menu items: drinks, desserts, and main courses.

### ✨ Key Features
* **Easy navigation:** Quickly switch between menu categories (drinks, desserts, main courses) without reloading the page.
* **Embedded QR code:** A QR code is placed on the page that leads to this website. Guests or staff can download it in PNG format directly from the screen to print and place on tables.  * **Responsive design:** The menu displays equally beautifully and clearly on both computer screens and smartphones.
* **Maximum speed:** The site runs entirely client-side (Vanilla JS), so tab switching is instantaneous.

### 🛠 Tech Stack
* **HTML5 / CSS3** — clean layout and intuitive interface.
* **Vanilla JavaScript (ES6+)** — menu tab switching logic and interface management.
* **QRCode.js** — a library for displaying a QR code for a website link on the client-side.
